# 🚀 Установка и использование Whisper на Windows

Скрипты и инструкция для быстрой установки **Whisper** — нейросетевой модели распознавания речи от OpenAI.

## ✨ Возможности
0
- 🎤 Распознавание речи из аудио- и видеофайлов
- 🌍 Поддержка множества языков (включая русский)
- ⚡ Работа на CPU и GPU (ускорение с CUDA)
- 💻 Простая установка и использование через командную строку

---

## Способ 1. Автоматическая установка (рекомендуется)

Если в проекте есть файл **`install_whisper.py`**, просто запустите его:

```bash
python install_whisper.py
```

Скрипт автоматически установит необходимые компоненты и подготовит окружение.

---

## Способ 2. Ручная установка

## 1. Установите Python

Скачайте Python версии **3.10–3.12** с официального сайта:

👉 https://www.python.org/downloads/windows/

Во время установки обязательно поставьте галочку **"Add Python to PATH"**.

Проверьте установку:

```bash
python --version
```

---

## 2. Установите FFmpeg

FFmpeg необходим для обработки аудио и видео.

### 📥 Шаг 1. Скачайте FFmpeg

Перейдите на сайт:

👉 https://www.gyan.dev/ffmpeg/builds/

В разделе **Release Builds** скачайте:

- `ffmpeg-release-full.7z` *(рекомендуется)*

---

### 📂 Шаг 2. Распакуйте архив

Установите **7-Zip**, если он отсутствует.

Распакуйте архив, например, в:

```text
C:\Tools\ffmpeg
```

После распаковки структура будет выглядеть примерно так:

```text
C:\Tools\ffmpeg\
└── ffmpeg-<версия>\
    └── bin\
        └── ffmpeg.exe
```

---

### ⚙️ Шаг 3. Добавьте FFmpeg в PATH

1. Откройте:

   **Панель управления → Система → Дополнительные параметры системы**

2. Нажмите **Переменные среды**.

3. В разделе **Системные переменные** выберите **Path** → **Изменить**.

4. Нажмите **Создать** и добавьте путь до папки **bin**, например:

```text
C:\Tools\ffmpeg\ffmpeg-7.1.1-full_build\bin
```

5. Нажмите **ОК** во всех окнах.

---

### ✅ Шаг 4. Проверьте установку

Откройте новую командную строку и выполните:

```bash
ffmpeg -version
```

Если появилась информация о версии FFmpeg — установка выполнена успешно.

---

## 3. Установите Whisper

Выполните команду:

```bash
pip install git+https://github.com/openai/whisper.git
```

Если появляется ошибка, убедитесь, что установлен **Git**.

---

## 4. (Необязательно) Установите PyTorch с поддержкой CUDA

Если у вас есть видеокарта NVIDIA, можно использовать GPU для ускорения работы.

Пример установки:

```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

Если используется другая версия CUDA, замените `cu118` на соответствующую (например, `cu121`).

Проверить совместимость можно на официальном сайте PyTorch.

---

# ⚙️ Проверка установки

Выполните тестовое распознавание:

```bash
whisper audio.mp3 --model tiny
```

При первом запуске выбранная модель автоматически загрузится из интернета.

Если всё настроено правильно, в консоли появится распознанный текст.

---

# ▶️ Использование

Базовый синтаксис:

```bash
whisper <путь_к_файлу> [--model <модель>] [--language <язык>]
```

### Примеры

Распознавание русского языка:

```bash
whisper meeting.wav --model small --language Russian
```

Перевод речи и экспорт субтитров:

```bash
whisper video.mp4 --model medium --task translate --output_format srt
```

# ⚠️ Важно

- При первом запуске модель автоматически загрузится из интернета (примерно **1–3 ГБ**).
- Для работы на GPU необходимы драйверы NVIDIA и установленный CUDA Toolkit.
- Убедитесь, что Python и FFmpeg добавлены в переменную **PATH**.
- При ошибках `ffmpeg` проверьте правильность установки и перезапустите командную строку.
